■必須
●sudo [コマンド]
管理者としてコマンドを実効

●shutdown -h now
シャットダウン 管理者権が必要
Haltのh

●shutdown -r now
再起動 管理者権利が必要
Rebootのr

●apt-get update
●apt-get upgrade
●apt-get install [パッケージ名]
apt-getを使ってデータベースの更新、
更新されたソフトがあればバージョンアップ、パッケージのインストール
APricaTion-GET

●echo [文字列]
文字列をそのまま返す
ECHO（やまびこ）

■ディレクトリとファイル
●pwd
カレントディレトリ表示
Print Working Directory
（作業中のディレクトリを出力する、という意味）

●ls [ディレクトリパス]
指定したパスにあるディレクトリとファイルを表示
省略時はカレントディレクトリ
LiSt

	オプション
	 -l ファイルのみアクセス権や更新日時などを表示
	 -d ディレクトリも含めてアクセス権やタイムスタンプを表示
	 -a .で始まる隠しファイル、ディレクトリも表示

●mkdir [ディレクトリパス]
指定したディレクトリを作成
MaKeDIRectory

●cp [元ファイル] [先ファイル]
ファイルのコピー
CoPy

●mv [元ファイル] [先ファイル]
ファイルの移動とリネーム
MoVe

●rm [ファイルパス]
ファイルを削除
ReMove

●cat [ファイルパス]
指定されたテキストファイルの中身を表示する
本来は複数のファイルの中身をつなげて表示するためのものらしい
conCATenate

●chmod [対象]+/-[アクセス権] [ファイル/ディレクトリパス]
アクセス権の変更
User,Groupe,Otherのいずれかに対して対して+か-で
Read（読み込み）,Write（書き込み）,eXecute（実行）の属性を付けたりはずしたりできる
CHange MODe
